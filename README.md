# music_app

## Description
A basic music streaming app using flutter

## Features
- Email Password Authentication (Multiple Login)
- Play/Pause Songs
- Search songs
- Dark Mode / Light Mode
- Songs can be marked as Favourite
- Offline syncing
- No syncing when favourite screen is open
- Cool Hero animations

## Tools and Libraries
- This app has been created using flutter framework and firebase , for state management in flutter provider package is used.
